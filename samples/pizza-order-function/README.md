
# Pizza Order Function

A coded agent that prices and confirms pizza orders with quantity discounts, crust surcharges, and prep time estimates.

## Run

```
uipath run main '{"size": "large", "crust": "stuffed", "toppings": ["pepperoni", "mushrooms", "olives"], "quantity": 3, "customer_name": "Alice"}'
```

## Debug

```
uipath dev web
```

# UiPath Configuration Specifications

Complete specifications and JSON schemas for UiPath configuration files.

## Files

### `uipath.json` - Project Configuration
Runtime options, design settings, packaging rules, and Python function entrypoints.

- [Spec](uipath.spec.md) | [Schema](uipath.schema.json)

### `bindings.json` - Resource Bindings
Asset, process, bucket, index, and connection bindings.

- [Spec](bindings.spec.md) | [Schema](bindings.schema.json)

### `entry-points.json` - Entry Points
Entry point definitions with JSON Schema-based input/output contracts.

- [Spec](entry-points.spec.md) | [Schema](entry-points.schema.json)

### `project.uiproj` - Project Metadata
Project-level metadata for StudioWeb integration. Auto-generated and auto-updated by `uipath init`.

- [Spec](uiproj.spec.md) | [Schema](uiproj.schema.json)
